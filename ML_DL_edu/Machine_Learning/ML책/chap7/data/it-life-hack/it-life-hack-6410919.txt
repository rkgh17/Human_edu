Excel에서 지사 이름의 목록을 빠르게 연속 입력 [지 이득 · 치트 시트]
회사의 지사 이름이나 부서명, 취급 상품명 등 서류 중에 항상 정해진 목록을 입력하는 것은 자주있다. 이러한 연속적인 목록을 Excel에서 빠르게 입력 할 수 있도록 등록 해 두자. 서류 작성의 번거 로움을 크게 줄일 수있게된다.

■ 지 이득 No.288 사용자 지정 목록을 추가
Excel에서 연속 데이터를 입력하는 기능이있다. 예를 들어, "1 월"을 입력 한 셀의 오른쪽을 아래로 드래그하면 "2 월" "3 월"...... 같이 연속 된 데이터가 자동 입력된다 (그림 1).

 
그림 1 "1 월"을 입력 한 셀의 오른쪽 아래에 마우스 포인터를 거듭하고, "+"가 표시된 시점에서 아래로 드래그. 연속적인 데이터가 입력된다.

연속 데이터로 입력하는 목록 숫자와 달 이외에도 사용자가 새로 등록 할 수있다. 여기에서는 회사의 지사 이름을 등록 해 보자. 첫 건만 입력하면 후 드래그하면 목록을 입력 할 수있게된다.

● 옵션 화면에서 사용자 설정 목록 등록
목록은 "사용자 지정 목록"에 등록한다. 미리 목록을 만든 후 범위 지정 (화면 2) 옵션 화면에서 등록을 시작하려고 (화면 3-4). 등록이 완료되면 연속적인 목록으로 빠르게 입력 가능하게된다 (화면 5).

 
화면 2 미리 작성한 목록을 범위 지정한다.


 
화면 3 메뉴에서 [파일] → [옵션]을 선택합니다. 고급의 일반의 사용자 지정 목록 편집 버튼을 클릭한다.


 
화면 4 가져 오기 버튼을 클릭하여 선택 목록 항목이 표시되면 [추가] 버튼을 클릭합니다. 이 후 OK를 클릭하여 설정 화면을 닫습니다.


 
그림 5 "도쿄 본점"를 입력 한 셀의 오른쪽 아래에 마우스 포인터를 거듭하고, "+"가 표시된 시점에서 아래로 드래그. 연속적인 데이터가 입력된다.

등록 된 사용자 지정 목록은 새 파일을 만들 때 사용할 수있다. 자주 사용하는 목록을 미리 등록 해두면 도움이된다.
