또 하나 서비스가 사라진다! 도코모 web 편지 종료하는 이유
도코모는 2012 년 3 월 5 일 "도코모 web 메일」을 2012 년 8 월 27 일 오전 11시에 종료하는 것을 명확하게했다.
"도코모 web 편지」는 「i 모드 메일 자동 저장 '나'쉽게 로그인 '등의 i 모드 연계 기능을 갖춘 i 모드 · i 어플리 · PC에서 사용 가능한 WEB 메일 서비스로, 2010 년 4 월 14 일 더 제공했다.

서비스를 종료하는 이유이지만, 스마트 폰을 포함한 인터넷 시장의 변화에 ​​따라 경영 자원을 집중하기 위해 노력하고있다.

도코모는 서비스 종료에 앞서 2012 년 3 월 13 일 오전 11시를 가지고 '도코모 web 메일」의 신규 신청 접수를 종료한다.

사용자가 이전에주고받은 이메일 데이터의 저장 방법 등에 대해서는, 6 월 상순에 「도코모 web 메일에서 안내」에서 안내한다.

도코모는 앞으로도 사용자에게 더욱 나은 서비스를 제공하기 위해 노력하고 간다고하고있다.
