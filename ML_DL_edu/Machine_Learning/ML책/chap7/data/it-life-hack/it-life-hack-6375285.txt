창을 닫지 않고 바탕 화면 아이콘을 조작하는 극비 비결 [지 이득 · 치트 시트]
바탕 화면의 아이콘을 이용하고 싶지만, 현재 창에 숨어 버린 수있다. 그러나 창을 닫 않으려면 도구 모음에서 바탕 화면 아이콘을 조작하는 방법이 있기 때문에 소개하자.

■ 지 이득 No.283 "바탕 화면"도구를 활용하는
일반적으로 작업 표시 줄에 입력 도구 모음 도구 모음이 표시되어있다. 여기에 바탕 화면 아이콘을 표시하려면 도구 모음을 추가하는 방법이 있기 때문에 시험해 보자. 여기에서 아이콘을 선택하면 일일이 표시를 전환 수고는 필요 없어진다.

● "바탕 화면"모음을 표시하려면
작업 표시 줄에서 마우스 오른쪽 단추로 클릭하고 도구 모음] - [바탕 화면]을 선택 (화면 1). 이제 화면 오른쪽 하단에 데스크톱 도구 모음이 표시된다 (화면 2).

 
화면 1 작업 표시 줄에서 마우스 오른쪽 단추로 클릭하고 도구 모음] - [바탕 화면]을 선택한다.


 
화면 2 바탕 화면 도구 모음이 표시되었다.

바탕 화면 도구 모음의 오른쪽에있는 >>를 클릭하면 바탕 화면의 아이콘이 나열되어 거기에서 응용 프로그램을 열거 나 폴더 나 파일을 열 수 (화면 3).

 
화면 3 오른쪽의 [>>]를 클릭하면 바탕 화면의 아이콘이 표시된다.

이것이라면 데스크톱 화면으로 전환하지 않고 언제든지 아이콘이 유효하다. 게다가 라이브러리에 빠르게 액세스 할 수있다.
또한 바탕 화면에서 아이콘을 숨길 수 있어도 도구 모음을 이용하면 아이콘을 이용할 수있다. 데스크톱은 깨끗이하면서 자주 사용하는 아이콘을 사용할 수 있으므로 꼭 활용 해 주었으면한다.
