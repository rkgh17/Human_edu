Excel 스프레드 데이터의 합계를 계산없이 표시하는 방법 [지 이득 · 치트 시트]
Excel에서 어떤 셀의 합계를 알고 싶다면 어떻게해야하는 것일까? 총 값을 입력 할 셀이 있다면, 합계 수식을 입력 해두면 좋은 것이지만 테이블에있는 부분의 총 가치를 알고 싶은 경우 등이다. 사실 계산식 등 이용하지 않고 총 값을 알 수있는 방법이있다.

■ 지 이득 No.287 상태 표시 줄의 정보를 이용하는
여러 셀의 합계는 사실 특별히 설정하지 않아도 기본적으로 화면에 표시되도록되어있다. 그 장소가 상태 표시 줄이다. 총 외에도 평균도 알게되었다.

● 여러 파일을 선택하여 합계를 확인하는
총을 알고 싶은 셀을 선택 상태로 화면 하단의 상태 표시 줄의 오른쪽 부분을보고 싶어요. 평균 : ●● 데이터의 개수 : ●● 총 : ●●로 표시되는 것이다 (화면 1).

 
화면 1 셀을 선택하면 상태 표시 줄의 오른쪽에 합계가 표시됩니다.

무려 셀을 선택하면 해당 셀에 입력 된 숫자의 합계뿐만 아니라 평균과 선택한 셀의 개수까지 알 것이다. 계산기를 꺼내 계산하고, 만일 어딘가의 셀에 수식을 입력하고 계산하고보고하는 등의 번거 로움이 전혀 필요 없다.

상태 표시 줄에 표시 할 내용은 상태 표시 줄을 오른쪽 클릭하면 나타나는 상태 바의 사용자 설정에서 선택하여 변경할 수있다 (화면 2).

 
화면 2 상태 바의 사용자 설정에서 표시 할 내용을 지정할 수있다.

그때 그때하고 필요로하는 정보가 표시되도록 설정을 변경하면 좋을 것이다.
