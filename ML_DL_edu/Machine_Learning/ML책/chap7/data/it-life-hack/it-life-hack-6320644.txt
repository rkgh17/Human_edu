고사양의 그래픽을 탑재! 마우스 컴퓨터, 코스 파 뛰어난 고성능 게임 PC를 발매
마우스 컴퓨터 마우스 컴퓨터의 게이밍 브랜드 'G-Tune "보다 대용량 64GB 물건 메모리를 탑재 한 게임 PC를 출시했다.

최신 CPU와 고사양 그래픽을 탑재하면서도 뛰어난 코스트 퍼포먼스를 실현하는 「NEXTGEAR i830 시리즈 "의 대용량 메모리 탑재 모델로 인텔 Core i7-3820 프로세서, 대용량 쿼드 채널 64GB 메모리, 1TB 하드 디스크 (7200rpm ), DVD 슈퍼 멀티 드라이브, NVIDIA GeForce GTX580,80PLUS GOLD 인증 취득 700W 전원 공급 장치, Windows 7 Professional with SP1 64 비트 정품을 설치 한 상태 "NEXTGEAR i830GA1-SP"가 18 만 9,840 엔.

 
 

헤어 라인 가공을 한 고급스러운 용맹스러운 전면 패널이 특징 인 G-Tune의 걸작 "MASTERPIECE i1550 시리즈"의 대용량 메모리 탑재 모델로 인텔 Core i7-3820 프로세서, 대용량 쿼드 채널 64GB 메모리, 1TB 하드 디스크 (7200rpm), DVD 슈퍼 멀티 드라이브, NVIDIA GeForce GTX580를 2 장 탑재 한 SLI 구성 80PLUS 인증 취득 1200W 전원 공급 장치, Windows 7 Professional with SP1 64 비트 정품을 설치 한 상태 "MASTERPIECE i1550GA1-SP"가 27 만 9,930 엔.

 
 

앞으로도 G-Tune에서는 항상 최신 · 최고의 성능을 발휘하는 제품을 출시하고 사용자 "수 ·하고 싶다」를 실현하는 PC를 계속 공급하고있다.
