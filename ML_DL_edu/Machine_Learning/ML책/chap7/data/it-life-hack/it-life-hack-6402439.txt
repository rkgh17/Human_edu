최근에 사용한 파일을 빠르게 시작 메뉴를 통해 [지 이득 · 치트 시트]
Windows 7 시작 메뉴에 최근에 사용한 항목이 사라 졌어요. 항상 사용하고있어 편리이었는데 ......라고 생각하는 사람도있을 것. 사실이 항목 시작 메뉴에 표시되도록 설정할 수있다.

■ 지 이득 No.286 시작 메뉴에 최근에 사용한 항목을 표시하는
Windows 7에서는 최근 파일은 점프 목록에서 열 수도있다. 그러나 작업 표시 줄에 등록하지 않은 응용 프로그램의 파일 인 경우 일단 앱을 실행 한 후이 아니면 이용할 수 없다. 시작 메뉴에서 선택할 수 있도록 해두면, 이러한 경우에도 확 파일을 열 수있다.

● 속성을 사용자 정의 할
시작 단추를 마우스 오른쪽 단추로 클릭하고 속성을 선택한다. [사용자 정의]를 클릭하면 (화면 1) 목록에서 최근에 사용한 항목을 선택하고 OK를 클릭하자 (화면 2). 이제 설정은 완료되었다.

 
화면 1 [작업 표시 줄 및 시작 메뉴 속성에서 사용자 지정을 클릭한다.


 
화면 2 [최근에 사용한 항목을 선택한다.


시작 메뉴를 열어 보면 오른쪽에 최근에 사용한 항목이 표시되고 마우스 포인터를 합치면 최근에 사용한 파일 목록이 표시된다 (화면 3). 여기에서 선택하여 원하는 파일을 빠르게 여는 셈이다.

 
화면 3 시작 메뉴에서 최근에 사용한 항목을 이용할 수있다.


이것이라면 작업 표시 줄에 등록하지 않은 응용 프로그램의 파일을 여는 것은 물론, 각 앱에 각각 점프 목록을 표시하고는 선택이 아니라 같은 장소에서 열 수 있기 때문에 편리하다.
